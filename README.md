## Prerequisites
* [Node](https://nodejs.org/en/)

## Getting Started
Before run tests, run the command below in source folder
```
npm install
```

## Running tests
To run full tests
```
npm test
```

To run contract tests
```
npm run test-contract
```

To run integration tests
```
npm run test-integration
```
